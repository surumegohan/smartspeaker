### ボイスフィードバック

AmazonのAlexaスキル「ボイスフィードバック」の説明です。


このスキルではスキルの開発者がフィードバックを気軽に受けられる一助になることを目指しています。  
VUIのアプリケーションが「ストップ」で終わるだけでは悲しいです。
もし、「いいね」や「いまいち」でもスキルを終了できて、かつ、フィードバックを同時にできたら素敵だと思いました。  


### 利用規約

個人情報などは一切取得しておりません。  
  
もちろん無料なので、自由にお使いください。  
もし、このスキルで「なるほど！」となっていただければ幸いです。
